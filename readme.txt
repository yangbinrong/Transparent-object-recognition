此处代码为2019世界机器人大赛共融机器人挑战赛Task D中双臂协同搬运任务中，对半透明圆盘进行识别、定位、径向畸变补偿及障碍物识别、运动小球实时识别等视觉处理的部分源码。
image_process.cpp为自主编写的功能库，包含kinect相机调用、相片存储功能、根据标定矩阵进行图像坐标系到机器人坐标系映射的功能、获取桌面物体的抓取位置姿态功能、 获取机械臂上物体的模型名称（三棱柱、圆柱、方块）功能、 获取立方体底面的形心位置功能、堆叠物体识别功能、零件截面识别功能。
balltracking.cpp是实现半透明圆盘识别、定位、径向畸变补偿等视觉处理及障碍物识别、运动小球实时识别定位的源码，包括相机图片捕获、像素坐标系到世界坐标系的转换、轮廓识别、绘制轮廓、二次轮廓识别、径向畸变补偿、障碍物轮廓识别、基于颜色分割的运动小球实时识别小球、坐标发送等功能。
balltracking_test.cpp为测试用例。
